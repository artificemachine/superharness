## 1. Headless Architecture
The Guardian is now lean. No more "forced" UI.

*   **Operator engine:** Headless by default.
*   **Opt-in UI:** Use `--dashboard` to keep a persistent dashboard alive.
*   **On-Demand UI:** `shux dashboard` starts a temporary session.
*   **Lease Pattern:** Standalone dashboards automatically exit after a timeout (default 5 min if not interacting).

## 2. Test Suite Recovery
The systemic failures in CI were resolved:
*   **Heartbeat Mocking:** Tests now seed SQLite heartbeats for participants before starting discussions.
*   **DAO Fix:** `heartbeat_dao.py` was updated to support the v25 migration schema (was causing crashes in the new availability gates).
*   **Isolation:** `test_model_router.py` is now isolated from the local `fleet.yaml` config via automatic mocking.
